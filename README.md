# RepoLens – GitHub Repository Intelligence Analyzer

RepoLens is a Python tool that analyzes GitHub repositories and generates actionable insights about repository popularity and activity.

## Features
- Fetch repository data using GitHub API
- Display repository metrics: stars, forks, primary language, open issues
- Generate automated insights on repository popularity and activity
- Modular design separating data collection and analysis
